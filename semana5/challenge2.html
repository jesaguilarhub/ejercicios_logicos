Plantar Arboles

Alberto necesita ayuda para plantar algunos arboles. Para ello te da 3 parametros relacionados al campo:

  • Ancho (width) del campo w
  • Longitud (length) del campo l
  • Brecha (gap) entre arboles g

Tienes que crear un algoritmo que regrese el numero de arboles que pueden plantarse en los bordes del campo dado en un diseño simetrico mostrado a continuación (asimétrico gap = x, tree = o, gap = -)

Ejemplos:

1.-

w = 3, l = 3, g = 1
plantTrees(w, l, g) ➞ 4

o - o
-   -
o - o

2.-

w = 3, l = 3, g = 3
plantTrees(w, l, g) ➞ 2

o - -
-   -
- - o


Si el diseño no es simetrico, debes de regresar 0

w = 3, l = 3, g = 2
plantTrees(w, l, g) ➞ 0

o - -
x   o
x x x
// Plantar 2 arboles significa que la brecha de dos arboles sea mayor a 2.

o - -
x   o
o - -
// Plantar 3 arboles significa que la brecha de los dos arboles será menor a 2.


Otro ejemplo para mejor entendimiento

w = 3, l = 3, g = 0
plantTrees(w, l, g) ➞ 8

o o o
o   o
o o o

Notas:

  • Recuerda que los arboles se deben de plantar a los bordes del campo
