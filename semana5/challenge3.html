Encuentra las letras únicas

Crea una función que tome un string y devuelva un arreglo de las letras que aparezcan una sola vez

Ejemplos:

findLetters("monopoly")
Output = ["m", "n", "p", "l", "y"]

findLetters("balloon")
Output = ["b", "a", "n"]

findLetters("analysis")
Output = ["n", "l", "y", "i"]

Notas:

  • El arreglo final no debe incluir letras que aparezcan más de una vez en la cadena.
  • Devuelva las letras en la secuencia en la que estaban originalmente, no las ordenes.
  • Todas las letras deben de estar en minúsculas.
